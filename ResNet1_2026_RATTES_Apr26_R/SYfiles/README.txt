These files were prepared by Melissa Foster using the Sediment Yield model component 1. These are input files used for the MLR model.

MLR_Input_GRAINSIZE.csv = Input file used in MLR, derived from ResNet
MLR_SAatDamInitial_GRAINSIZE.csv = Sediment-contributing drainage area time series using as-built trap efficiency and capacity, which is a parameter in the MLR model

Grain sizes are silt, clay, and sand.
