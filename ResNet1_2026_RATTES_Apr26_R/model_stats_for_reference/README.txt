These files contain model summaries that are automatically generated by 1_MLR_modeltraining.R. They are included here for reference so that users can view the model results without running any code, although the files can also be reproduced by executing the script.

Supplementary Table 1 provides an overview of the statistics reported in each model summary. The Model IDs listed in Supplementary Table 1 correspond to the following files:

Model 1: lsr_summary_subset_best4_rattes_v1p2_silt.txt
Model 2: lsr_summary_subsetdata_pca_rattes_v1p2_silt.txt
Model 5: lsr_summary_subset_best4_rattes_v1p2_3param_silt.txt


Model summary files are not provided for the training-only models (Models 3,4, and 6). These models were used exclusively to evaluate predictive performance on the test dataset and were not used to generate future reservoir sedimentation predictions. The corresponding model outputs can still be accessed by running 1_MLR_modeltraining.R.
